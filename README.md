# 台北一日遊 - Taipei-day-trip

## ＃專案介紹

Website URL 網站連結 : [台北一日遊-Taipei-day-trip](http://35.83.22.153:3000/)

台北旅遊電商網站，串接台灣政府公開資源台北景點 api。
使用者可以在本網站做到以下幾點：

1. 瀏覽、關鍵字搜尋台北景點
2. 點擊景點可觀看該景點詳細資訊，包含交通方式、景點詳細介紹、地址
3. 登入會員後，可以預訂該景點旅遊行程以及測試信用卡付款功能
